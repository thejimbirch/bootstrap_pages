INTRODUCTION
------------

The Bootstrap Pages module will provide a suite of Content Types, Taxonomies, and Views to work with the Bootstrap Paragraphs module.

**Types:**

  * Article
  * Author
  * Landing Page

REQUIREMENTS
------------

This module requires the following modules:

  * bootstrap_paragraphs
  * breakpoint
  * ctools
  * field
  * field_group
  * image
  * imageapi_optimize
  * imagemagick
  * image_effects
  * inline_entity_form
  * media
  * metatag
  * metatag_open_graph
  * node
  * path
  * pathauto
  * responsive_image
  * system
  * taxonomy
  * views
  * Bootstrap framework's CSS and JS included in your theme

INSTALLATION
------------

  * Install the module as you normally would.
